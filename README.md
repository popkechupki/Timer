# Timer
Mechanics plugin for PocketMine-MP.
使用する際には以下のライセンスに同意した上での使用をお願いします。
## HowToUse
| Command | 実行可能な操作 | 実装したバージョン |
|:-----------|:------------:|:------------:|
| /timer start <秒数>  | <秒数>分のタイマーを起動        | 0.1 |
